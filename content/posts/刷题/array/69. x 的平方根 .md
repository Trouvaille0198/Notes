# 69. x 的 平方根 

`easy`

## 二分法

不多 bb

```go
func mySqrt(x int) int {
	low, high := 0, x
	for low <= high {
		mid := low + (high-low)>>1
		if mid*mid < x {
			if (mid+1)*(mid+1) > x {
                // 找到最后一个小于x的mid
				return mid
			}
			low = mid + 1
		} else if mid*mid > x {
			high = mid - 1
		} else {
			return mid
		}
	}
	return -1
}
```

```go
func mySqrt(x int) int {
    if x == 0 {
        return 0
    }
    ans := int(math.Exp(0.5 * math.Log(float64(x))))
    if (ans + 1) * (ans + 1) <= x {
        return ans + 1
    }
    return ans
}
```
