基本操作

安装

1

npm install yarn

初始化一个新项目

1

yarn init

安装所有依赖项

1
2

yarn
yarn install

添加依赖项

1
2
3

yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]

将依赖项添加到不同的依赖类别中

1
2

yarn add [package] --dev  # dev dependencies
yarn add [package] --peer # peer dependencies

更新依赖项

1
2
3

yarn up [package]
yarn up [package]@[version]
yarn up [package]@[tag]

删除依赖项

1

yarn remove [package]

更新 Yarn 本体

1
2

yarn set version latest
yarn set version from sources

与 vite 配合

运行

1

yarn dev
