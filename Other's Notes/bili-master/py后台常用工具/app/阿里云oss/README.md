## 安装SDK

  ```bash
  pip install oss2			
  ```

## config文件参数说明

```
OssAccessKeyID = "你的AccessKeyID"
OssAccessKeySecret = "你的AccessKeySecret"
OssBucketName = "你的桶名字"
OssEndpoint = "你上传的地区端点" # 例如 "http://oss-cn-shanghai.aliyuncs.com"
OssHttps = "访问对应文件的链接组合"  # 例如 "https://xxxxxx.oss-cn-shanghai.aliyuncs.com/"
```
